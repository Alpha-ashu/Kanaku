# Wireframe Image Placeholders

The template references binary image files which cannot be authored as text:
- `Login_Wireframe.png`
- `Dashboard_Wireframe.png`

## How to add them
1. Design in **Figma** or **draw.io**.
2. Export as PNG with the exact filenames above.
3. Drop the PNGs into this `Wireframes` folder.

Until then, see `Flow_Explanation.md` for the textual wireframe descriptions,
and `../Sequence_Diagrams.md` for interaction flows.

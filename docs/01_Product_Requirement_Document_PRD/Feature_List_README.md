# Feature_List

The original template referenced `Feature_List.xlsx`. To keep documentation
**version-control friendly** (per best practices), the feature list is maintained
as `Feature_List.csv` in this folder.

- Open `Feature_List.csv` directly in Excel / Google Sheets / LibreOffice.
- To export an `.xlsx`: open the CSV in Excel and "Save As" → Excel Workbook.
- Columns: `ID, Feature, Module, Priority, Status, Owner, Acceptance_Ref, Notes`.
